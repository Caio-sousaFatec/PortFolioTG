<p align="justify">
O DataFlow é uma aplicação web para otimizar a configuração de fontes de dados em pipelines de processamento em cascata. Permite cadastro de clientes, upload e validação de dados, mapeamento de campos e aplicação de regras de negócios, tudo com dashboard analítico completo.
</p>

<summary align="center">VER MAIS DETALHES</summary>

#### Objetivo

- Cadastrar e gerenciar clientes, soluções e usuários autorizados.  
- Fazer upload de dados com visualização e validação de estrutura (campos, tipos, regras de negócios).  
- Aplicar regras de negócios (mapeamento "de/para").  
- Disponibilizar dashboard analítico com métricas de configuração.  
- Permitir acompanhamento completo via interface analítica.

#### Tecnologias Utilizadas

- **Java JDK 21.0.2**: Backend.  
- **Spring Boot 3.3.2**: Framework backend.  
- **MySQL**: Banco de dados.  
- **HTML5/CSS3/JavaScript**: Frontend.  
- **Vue.js 2.6.12**: Framework frontend.  
- **ClickUp**: Gerenciamento do projeto.  
- **Figma**: Protótipos.  
- **IntelliJ IDEA**: IDE backend.  
- **Visual Studio Code**: IDE frontend.  
- **Postman**: Teste e documentação de APIs.
